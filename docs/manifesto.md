# Manifesto Nika

**Versão 0.1 — Março 2026**

---

## O que é o Nika

O Nika é uma plataforma social e comunitária de código aberto.

Não é uma rede de influência. Não é um produto de engajamento. Não tem algoritmo que empurra conteúdo para te manter scrollando.

O Nika é feito por coletivos, para coletivos.

---

## Premissas

**1. Tecnologia é política.**
Toda escolha técnica carrega uma visão de mundo. Escolhemos código aberto, federação futura, e estruturas horizontais. Isso não é decoração ideológica — é arquitetura.

**2. Engajamento não é aprovação.**
Redes sociais comerciais quantificam atenção como valor. O Nika rejeita isso. Reações são ações: amplificar, traduzir, desafiar, documentar. Não há contador de likes.

**3. Comunidade é mais que grupo.**
Comunidade no Nika é um espaço de organização real — com curadoria, história e decisões coletivas. Não é uma lista de membros em torno de um hashtag.

**4. Conhecimento é ponto de vista.**
O Acervo do Nika não finge neutralidade. Toda página tem contexto, autor, e perspectiva explícita. Isso é mais honesto do que neutralidade simulada.

**5. O Nika é um hub, não uma ilha.**
A plataforma conversa com outros PWAs, ferramentas e projetos. Federação real via ActivityPub está planejada para fase posterior, mas a mentalidade de ecossistema existe desde o dia um.

---

## O que o Nika não é

- Não é uma rede de influenciadores.
- Não é monetizado por atenção.
- Não tem publicidade.
- Não é neutro politicamente.
- Não é completo — e nunca "estará pronto".

---

## Identidade ideológica no produto

O posicionamento anarco-comunista do Nika não é apenas um manifesto README.

Ele aparece nas decisões de produto:

| Escolha técnica | Raiz ideológica |
|---|---|
| Reações como ações, não likes | Rejeição de métricas de aprovação individual |
| Moderação por consenso (planejada) | Horizontalidade, sem donos de comunidade |
| Código aberto por padrão | Propriedade coletiva do trabalho |
| Sem algoritmo de feed | Autonomia de quem consome |
| Federation futura | Descentralização real, não cosmética |
| Acervo com ponto de vista explícito | Honestidade sobre perspectiva |

---

## Sobre o nome

Nika (νίκα) é uma palavra grega que significa vitória — usada na revolta de Nika em 532 d.C., quando o povo de Constantinopla se levantou contra o Estado.

É uma referência histórica a coletivos que resistem. É também simples, curto e funciona como domínio.
